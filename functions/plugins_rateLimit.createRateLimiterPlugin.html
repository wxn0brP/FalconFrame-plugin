    Function createRateLimiterPlugin

    • createRateLimiterPlugin(opts: RateLimiterOptions): Plugin

      Creates a rate limiter plugin for HTTP requests based on IP.

      Uses a simple "fixed window counter" algorithm and can optionally share the internal Map or disable cleanup interval.

      Parameters

      • opts: RateLimiterOptions

      Returns Plugin

      Middleware plugin compatible with your system.

      Example

      const sharedMap = new Map();
      const rateLimiter = createRateLimiterPlugin({
        maxRequests: 5,
        windowMs: 60_000,
        sharedMap,
        onLimitReached: (req, res, ctx) => {
          res.statusCode = 429;
          res.json({
            message: "Sorry, too many requests!",
            retryAfter: ctx.retryAfter,
            remaining: ctx.remainingRequests,
            ip: ctx.ip,
          });
        },
      });
